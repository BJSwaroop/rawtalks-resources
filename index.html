Action Plan
JoSAA Counselling Guide
Step-by-step process. Ee 6 steps follow cheyi.
  1. Register on JoSAA Portal josaa.admissions.nic.in — June lo opens. JEE Main score tho NITs/IIITs/GFTIs ki apply cheyyochu.
  2. Fill 50-80 Choices Dream + safe options mix cheyi. Order matters — preferred first! HS and OS options rendu add cheyi.
  3. Lock Before Deadline Last date ki 1 hour mundhu lock cheyi — server crash risk. "Float" use cheyi for upgrades.
  4. Document Verification Class 10/12 marks, category cert, rank card, Aadhaar, photos. Physically go to reporting center.
🔄
The 3 Options Explained

Float: Current seat keep + higher preferences lo upgrade try. Use this in Round 1-3.

Slide: Same institute lo better branch ki upgrade. Great for branch change.

Freeze: Final decision. Only use when 100% sure. NEVER in early rounds.

Strategy

Round 1-3: Always "Float." Seats open as students withdraw — later rounds lo rank shifts significantly. JoSAA has 5-6 rounds. Patience pays off.
